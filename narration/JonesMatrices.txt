From Bennett Section 6.5.1: each optical element such as a linear polariser or wave plate is represented in the Jones notation by a two-by-two matrix M. If E-tilde-i is the incident Jones vector, the transmitted Jones vector is E-tilde-t equals M times E-tilde-i. This is Bennett Equation 6.22.

Matrix multiplication is not commutative. For a sequence of N elements numbered 1 to N where element 1 is encountered first, the total Jones matrix is M-N times M-N-minus-one times ... times M-2 times M-1. The element encountered first goes on the right. This is Bennett Equation 6.23. If you reverse the order you get the wrong answer.

Bennett's Table 6.1 gives the Jones matrices for all standard optical elements. The horizontal linear polariser has 1, 0 in the top row and 0, 0 in the bottom. The vertical polariser has 0, 0 top and 0, 1 bottom. The transmission axis at plus 45 degrees gives one-half times the matrix 1, 1, 1, 1. The transmission axis at minus 45 degrees gives one-half times 1, minus 1, minus 1, 1.

The quarter-wave plate with fast axis horizontal has matrix 1, 0, 0, i. Why i? The fast axis is horizontal, so the x-component travels faster — it accumulates less phase delay through the plate. The y-component travels more slowly along the slow axis and picks up an additional phase of pi over 2, which appears as the factor i equals e to the i pi over 2 in the lower right element.

The quarter-wave plate with fast axis vertical has matrix 1, 0, 0, minus i. The x-component is now on the slow axis and picks up the extra phase. The y-component is on the fast axis and acquires no additional phase relative to x.

The rotation matrix rotates a Jones vector through angle alpha: cosine alpha, minus sine alpha in the top row and sine alpha, cosine alpha in the bottom. Bennett derives this in Example 6.5 by requiring the matrix to map cosine theta, sine theta to cosine of theta plus alpha, sine of theta plus alpha and solving for the matrix elements using the angle-addition identities for cosine and sine.

Let's work through Bennett's Example 6.6 in full. Light polarised at 45 degrees — Jones vector 1, 1 divided by root 2 — is incident on a vertical linear polariser. The result is 0, 0; 0, 1 times 1, 1 over root 2, which equals 0, 1 over root 2. This is the normalised Jones vector for vertical polarisation with amplitude 1 over root 2. The fraction of intensity transmitted is 1 over root 2 squared equals one half. This confirms Malus's law: cosine squared of 45 degrees equals one half.

Now the same 45-degree incident beam through a quarter-wave plate with horizontal fast axis. The matrix 1, 0, 0, i times 1, 1 over root 2 equals 1, i over root 2. This is left circular polarisation. The quarter-wave plate converts 45-degree linear to left circular — none of the intensity is lost in a wave plate.

Now Bennett's Example 6.7, which is the most involved: 45-degree linear light through a half-wave plate with horizontal fast axis, then a quarter-wave plate with vertical fast axis, then a linear polariser at minus 45 degrees. There is no half-wave plate in Table 6.1, but a half-wave plate is two quarter-wave plates in series. The calculation requires multiplying four matrices and then a column vector.

Working from right to left: two QWPs with horizontal fast axis give the half-wave plate matrix 1, 0, 0, minus 1. Multiplying by the QWP with vertical fast axis 1, 0, 0, minus i gives the effective combination matrix 1, 0, 0, i — a QWP with horizontal fast axis. This means a half-wave plate followed by a QWP of opposite orientation gives the same effect as a single QWP. Finally, multiplying by the minus-45 polariser matrix and the incident Jones vector gives the result: one half of e to the minus i pi over 4, times 1, minus 1 over root 2. This is a wave of one-half the incident irradiance, linearly polarised at minus 45 degrees, with an overall phase factor e to the minus i pi over 4 that has no measurable consequence.

The rotation of a Jones matrix is given by M-prime equals R of alpha times M times R of minus alpha. This is Bennett Equation 6.24. Using this, you can derive the Jones matrix for a linear polariser at any angle from the horizontal or vertical matrix in the table, without memorising additional matrices.
