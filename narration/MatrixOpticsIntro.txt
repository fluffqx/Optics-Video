Matrix optics represents any paraxial optical system as a single two-by-two matrix. A ray is described by a two-component column vector containing its height y above the optical axis and its angle theta to the axis. Each optical element maps the input ray vector to the output ray vector via matrix multiplication.

The power of this method comes from the fact that multiplying matrices corresponds to cascading optical elements. A system of ten lenses, five free-space propagation sections, and three mirrors — which would require solving fifteen separate geometric equations — reduces to multiplying fifteen two-by-two matrices. Modern optical design software uses exactly this method for tracing millions of rays simultaneously.

The key constraint is that all matrices in this framework have a determinant of exactly one. This is a consequence of conservation of the optical path and is a useful check on matrix algebra.
