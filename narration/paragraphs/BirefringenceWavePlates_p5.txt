A quarter-wave plate is designed so that d times the absolute value of n-e minus n-o equals lambda-naught over 4, as given in Bennett Equation 6.10. The relative phase shift is then pi over 2. The minimum thickness for quartz at 589 nanometres is, from Bennett Example 6.3: d equals lambda-naught divided by 4 times the absolute value of n-e minus n-o equals 589 nanometres divided by 4 times the absolute value of 1.5533 minus 1.5442 equals 16.2 micrometres. Such thin plates are fragile, so a more practical thickness uses a higher value of m — Bennett Example 6.3 gives d equals 1.30 centimetres for m equals 200, which provides the same quarter-wave phase shift but is much more robust.
